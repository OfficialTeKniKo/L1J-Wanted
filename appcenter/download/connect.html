Connection Error FAQ

Windows Defender
  • Sometimes the Windows Defender program gives a false positive for a virus in the Launcher. To avoid this, during the installation of {SERVER_NAME} Launcher an exclusion is added to the windows registry so that this does not happen. If this still happens and Windows Defender removes the Launcher, add an exclusion yourself to the folder where Lineage is installed and run {SERVER_NAME} setup again. To add the folder exclusion do the following:

    ※ Select Start.
    ※ Open Settings.
    ※ Under Privacy & security, select Virus & threat protection.
    ※ Under Virus & threat protection settings, select Manage settings.
    ※ Under Exclusions, select Add or remove exclusions.
    ※ Select Add an exclusion, and then select from folders.
    ※ Write the folder where you have installed Lineage.

Microsoft Redistributables error
  • Redistributables Download
    The Lineage Warrior launcher has a built-in utility to check if the system has already installed the Microsoft redistributables necessary to launch Lineage. If not, they will be downloaded and installed automatically, but if for some reason the Lineage Warrior launcher is not able to perform this action, follow the steps below:

    ※ Click the "Redistributables Download" button above this text to download the necessary redistributables (compressed in a zip file)
    ※ Once the file is downloaded, unzip it using your favorite ZIP decompressor.
    ※ Double click on the file "VC_redist.x86.exe" to open the Microsoft installer and follow the steps indicated.
    ※ Double click on the "vcredist_x86" file to open the Microsoft installer and follow the steps indicated.
    ※ Run the Lineage Warrior to enjoy the world of Aden.
