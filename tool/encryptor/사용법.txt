#############################################
※ 블로그 URL 암호화(내부 APPCENTER 연계)
#############################################
※ 생성된 암호화 URL은 인게임과 연동 후 접속 정보 취득
#############################################
※ 키 변경 불요
※ 입력예시 : APP IP : 127.0.0.1 / APP PORT : 8080 / APP URI : /outgame/info
#############################################

1. Encryptor.exe 실행

2. APP IP(앱센터 외부아이피) / APP PORT(앱센터 포트번호) / APP URI(호출 URI) 입력

3. 암호화 버튼 클릭

4. 하단에 생성된 암호화 URL 복사

5. 접속기 세팅 블로그에 등록
